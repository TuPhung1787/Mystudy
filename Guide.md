# Guide for Express JS

## Start the initialize project

+ Step 1: install `npm init`
+ Step 2: `npm i express`
+ Step 3: create file `server.js`
+ Step 4: `npm install nodemon -g`
+ Step 5: `npm install body-parser`
+ Setp for test mongodb:
